# Task 2 - Cloud Monitoring and Alerts using AWS CloudWatch

## Objective
The objective of this task is to monitor cloud resources and configure alerts using AWS CloudWatch.

---

## Services Used
- AWS EC2
- AWS CloudWatch
- AWS SNS

---

## Steps Performed

### 1. Created EC2 Instance
- Launched an Amazon Linux EC2 instance.
- Used Free Tier eligible instance type.

### 2. Configured CloudWatch Monitoring
- Opened CloudWatch metrics.
- Monitored EC2 instance performance.

### 3. Metrics Monitored
- CPUUtilization
- NetworkIn
- NetworkOut
- DiskReadOps
- DiskWriteOps

### 4. Created Dashboard
- Created CloudWatch dashboard.
- Added graphs for monitoring metrics.

### 5. Configured Alarm
- Created CloudWatch alarm for CPU usage.
- Alert condition:
  - CPU Utilization > 70%

### 6. Notification Setup
- Configured SNS notification for alerts.

---

## Output
Successfully implemented cloud monitoring and alert system using AWS CloudWatch.
