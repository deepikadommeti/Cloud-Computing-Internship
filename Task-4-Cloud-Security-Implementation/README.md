# Cloud Security Implementation

## Objective
Implement cloud security by configuring IAM policies, secure storage, and data encryption on AWS.

## Services Used
- AWS IAM
- Amazon S3

## Steps Performed
1. Created an IAM user named `cloud-security-user`.
2. Attached the `AmazonS3ReadOnlyAccess` policy.
3. Used an Amazon S3 bucket for secure storage.
4. Uploaded files to the S3 bucket.
5. Enabled Server-Side Encryption (SSE-S3).
6. Enabled Block Public Access for the bucket.

## Security Features
- IAM Access Control
- Secure S3 Storage
- Server-Side Encryption (SSE-S3)
- Block Public Access

## Outcome
Successfully implemented cloud security using AWS IAM and Amazon S3.
